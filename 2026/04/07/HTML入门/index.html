基本代码标签

上一节我们看到了一个最简单的 HTML 文件长什么样。这一节来拆解一下这些“基本代码标签”，理解每个部分的作用。

  • 文档类型声明 <!DOCTYPE html>
    写在最前面，告诉浏览器这个文件使用的是 HTML5 标准。没有它，某些浏览器可能会用“怪异模式”解析，导致页面变形。

  • <html> 根标签
    整个 HTML 文档的根元素，所有内容都要包在它里面。一般会加上一个 lang 属性声明语言，比如 <html lang="zh-CN"> 表示中文网页。

  • <head> 头部标签
    头部的信息不会直接显示在页面主体中，但非常重要。常见的子标签有：

    • <meta charset="UTF-8"> —— 定义字符编码，避免乱码。
    • <title>页面标题</title> —— 浏览器标签页上显示的标题。
    • <link> —— 引入外部 CSS 文件。
    • <script> —— 引入或书写 JavaScript 代码。

  • <body> 身体标签
    用户真正看到的网页内容都放在这里。文字、图片、按钮、表格……全都在 <body> 里。

常用内容标签

标题标签 <h1> ~ <h6>

用来表示文章的层级标题。<h1> 最重要，一个页面建议只出现一次；<h2> 是副标题，依次往下递减重要性。

段落标签 <p>

用来包裹一段文字，浏览器会自动在段落前后添加间距。

换行标签 <br>

这是一个自闭合标签（不需要结束标签）。想让文字在哪里强制换行，就插一个 <br>。

水平线标签 <hr>

也是自闭合标签，会在页面中画一条横线，常用来分隔内容。

注释 <!-- 注释内容 -->

注释不会在页面上显示，但可以在源码中看到。用来给自己或团队写说明。

列表：让信息更有条理

HTML中有两种常用的列表：无序列表和有序列表。

  1. 无序列表 <ul>
    适合表示没有先后顺序的一组项目，比如购物清单、特点介绍等。每个列表项用 <li> 包裹。

浏览器显示效果：

  • 苹果
  • 香蕉
  • 橙子
  1. 有序列表 <ol>
    适合表示有顺序或排名的一组项目，比如操作步骤、排行榜。同样用 <li> 包裹每个项目。

浏览器显示效果：

  1. 打开浏览器
  2. 输入网址
  3. 按下回车

小技巧：你可以通过CSS修改列表前面的标记样式，比如把无序列表的圆点改成方块，或者把有序列表的数字改成罗马数字。后面学到CSS时再详细介绍。

链接：让页面“活”起来

链接（也叫超链接）是HTML最伟大的发明之一——它让全世界的信息能够相互连接。用 <a> 标签来创建链接。

  • href 属性：指定跳转的目标地址（URL）
  • 标签之间的文字：就是页面上显示的可点击文本

图片：让页面更生动

用 <img> 标签在网页中插入图片。这是一个自闭合标签，没有结束标签。

基本语法:

  • src 属性：图片的路径（可以是网络图片的URL，也可以是本地相对路径）
  • alt 属性：当图片无法显示时，会显示这段文字；同时它对搜索引擎和无障碍阅读（屏幕阅读器）很重要
